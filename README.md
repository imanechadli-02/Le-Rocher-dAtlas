## ✨ Aperçu

Site vitrine haut de gamme avec design sombre et accents dorés, animations fluides et expérience utilisateur soignée.

## 🚀 Stack Technique

| Technologie | Usage |
|---|---|
| **Next.js 14** (App Router) | Framework React SSR |
| **TypeScript** | Typage statique |
| **Tailwind CSS** | Styles utilitaires |
| **Framer Motion** | Animations au scroll |
| **GSAP** | Parallax hero & particules |
| **Lucide React** | Icônes |

## 📦 Installation

```bash
git clone https://github.com/<username>/rocher-atlas.git
cd rocher-atlas
npm install
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000)

## 🗂️ Structure

```
rocher-atlas/
├── app/
│   ├── layout.tsx        # Metadata SEO + fonts
│   ├── page.tsx          # Page principale
│   └── globals.css       # Styles globaux
└── components/
    ├── Navbar.tsx         # Navigation sticky + mobile
    ├── Hero.tsx           # Hero parallax GSAP + particules
    ├── About.tsx          # Section à propos + stats
    ├── Menu.tsx           # Menu interactif par catégorie
    ├── Gallery.tsx        # Galerie photo + lightbox
    ├── Reviews.tsx        # Avis clients Google
    ├── Contact.tsx        # Formulaire + carte + infos
    └── Footer.tsx         # Pied de page
```
